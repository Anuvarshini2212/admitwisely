# 🎓 AdmitWisely

AdmitWisely is a full-stack college discovery and comparison platform built using Next.js, Prisma, PostgreSQL, and Tailwind CSS.

The platform helps students explore colleges, compare placements and fees, save favorite colleges, and make smarter admission decisions through a modern responsive interface.

---

## 🚀 Features

- 🔍 Search colleges dynamically
- 🏫 Explore detailed college pages
- ⚖️ Compare colleges
- ❤️ Save favorite colleges
- 🔐 Authentication UI (Login / Signup)
- 📱 Fully responsive design
- 🌐 Dynamic API routes
- 🗄️ PostgreSQL database integration
- ⚡ Prisma ORM integration
- 🎨 Modern UI inspired by Careers360 & Collegedunia

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/admitwisely.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file:

